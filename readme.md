# 项目介绍

本项目是一个基于STM32H747 微控制器的TinyML项目，用于开展TinymL相关的实验和应用。
包括执行时间的监测、功耗监测和内存占用的评估等。

## 1. 基础功能
### 1.1 基于STM32H747 微控制器的多层次存储系统

### 1.2 基于STM32H747 微控制器的图像数据采集系统

### 1.3 基于STM32H747 微控制器的音频数据采集系统

### 1.4 基于STM32H747 微控制器的无线通信系统

### 1.5 基于STM32H747 微控制器的加速度传感器采集系统

## 2. 应用层面的探索
### 2.1. 基于TinyMLPT的图像分类
### 2.2. 基于TinyMLPT的运动监测
### 2.3. 基于TinyMLPT的语音识别



## 3. 辅助工程目录
### 3.1  BatchProject
用来批量编译和下载项目的脚本，包含以下功能：
1. 编译项目
2. 下载项目到开发板
3. 自动检查编译结果
基于UV4的命令行工具，支持Windows（x86和x64）


### 3.2 PythonProject
用来进行单元测试的Python脚本，包含以下功能：
1. Wifi_socket.py 用来测试开发板的Wifi连接和数据传输
2. camera_test.py 用来测试OV5640相机的寄存器读写和图像可视化
3. accelator_test.py 用来测试加速度传感器的读取和可视化
4. audio_test.py 用来测试音频传感器的读取和可视化
